<center>
	<h1>Object.toHtml</h1>
	<p>A simple package that let's you to convert Object into HTML!</p>
</center>

## How to use this
```javascript
let obj = {
	html: { 
	  head: { 
	    title: 'Hello world!' 
	  }, 
      body: { 
	    h1: 'Hello world!' 
	  } 
	}
}

obj.toHtml(process stdin);
// <!doctype html><html><head><title>Hello world!</title></head><body><h1>Hello world!</h1></body></html>
```
